### 功能

- [x] HTML/CSS/JS 代码编辑
- [x] 代码格式化
- [x] HTML编辑器支持emmet
- [x] 代码提示
- [x] Sublime 快捷键



### 使用到的框架

-   [React](https://reactjs.org/)
-   [CodeMirror](http://codemirror.com)
-   [Bulma.css](https://bulma.io/)
